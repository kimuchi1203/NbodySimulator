N-body simulation
for Android 2.3.3
using jni

step for run:
1. ndk-build
2. build & run on eclipse

if you cannot use jni/ndk-build,
you can modify source code.
change ndk-xxx() to xxx() and build on eclipse.

access EXTERNAL_STORAGE ("~/NbodySimulator/data.txt") to store initial condition.
by modify data.txt, user can change condition of simulation.

.apk is already released in android market.
https://market.android.com/details?id=com.NbodySimulator
